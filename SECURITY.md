# Security Policy

## Supported Versions
Only the latest stable version of AIDF is supported with security updates.

## Reporting a Vulnerability
Please report security issues responsibly via email:
security@aidf.dev

Do not open public issues for vulnerabilities.

## Security Measures
- Role-Based Access Control (RBAC)
- Encrypted Secrets Vault
- Agent Sandbox Permissions
- Immutable Audit Logs

## Responsible Disclosure
We appreciate responsible disclosures and will respond within 72 hours.
